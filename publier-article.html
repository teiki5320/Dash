⚙️ Configuration Supabase (à remplir une fois)

Récupère service_role key dans Supabase → Project Settings → API. Cette clé donne tous les droits sur la DB Kultiva, garde-la pour toi. Elle est stockée en local (localStorage), tu peux la vider avec le bouton ci-dessous.
